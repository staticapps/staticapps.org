--- layout: default title: Static Web Apps - A Field Guide ---

Static Web Applications are architected to run independently of the need for server-side dynamic processing. Static web architecture can ease common web development headaches and offers many great benefits. This site is designed to introduce newcomers to the world of static web applications and offer solutions to common challenges encountered while building them. Welcome!
